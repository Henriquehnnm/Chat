## 🚀 Começando

### Pré-requisitos

- Python 3.8+
- Git
- Dependências listadas em `requirements.txt`

### Instalação

1. **Clone o repositório**
   ```bash
   git clone https://github.com/Henriquehnnm/Chat.git
   cd Chat
   ```

2. **Instale as dependências**
   ```bash
   pip install -r requirements.txt
   ```

3. **Execute o aplicativo**
   ```bash
   python src/main.py
   ```
